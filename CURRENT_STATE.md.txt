Diagnosis:
Some runtime loader / preload system / lazy registry is attempting to fetch TSX source files through HTTP instead of relying on Vite chunk resolution.

This is NOT a build issue anymore.

dist/assets/DistributedRuntimeErrorBoundary-*.js

identify and remove invalid TSX fetch/preload behavior.
