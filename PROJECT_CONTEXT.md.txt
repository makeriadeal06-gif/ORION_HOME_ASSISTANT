# ORION — PROJECT_CONTEXT.md

## PROJECT OVERVIEW

ORION is an Android-first distributed cognitive home assistant platform inspired by adaptive AI systems such as Jarvis/Vy, but designed with its own architecture, identity and operational runtime.

The project combines:

* distributed runtime orchestration
* realtime device synchronization
* MQTT ecosystem integration
* Google Home ecosystem integration
* TriggerCMD desktop automation
* cognitive runtime orchestration
* adaptive contextual behavior
* multi-node synchronization
* production-ready runtime recovery

ORION is NOT a simple frontend assistant anymore.

It is currently evolving into:

* a distributed ecosystem runtime
* a realtime automation platform
* a contextual cognitive orchestration layer

---
